# 07 战斗系统

## 战斗定位

首版采用回合制战斗。战斗主轴是 **普通攻击、防御、躲避和 4 个职业技能**，状态卡只作为战斗中的临时修饰能力。玩家即使没有抽到理想状态卡，也必须能依靠基础行动和技能完成完整战斗。

状态卡可以提高伤害、提供确定暴击、提供确定躲避、提供减伤或提高职业资源获取，但不应直接替代核心行动。状态卡不是局外卡组，只在战斗中临时抽取和持有。战斗结算不使用直接躲避概率或直接暴击概率；概率只用于状态卡抽取，例如抽到暴击状态卡的基础概率为 5%。

## 基础参数

| 参数 | 默认值 |
| --- | ---: |
| 每回合行动力 | 3 点 |
| 每回合状态卡抽取 | 2 张 |
| 状态卡持有上限 | 5 张 |
| 每回合状态卡使用上限 | 2 张 |
| 每场战斗初始护甲 | 0 |
| 状态卡抽取配置 | 每场战斗开始生成，战斗结束清空 |

## 核心行动

玩家每回合都可以使用以下固定行动。固定行动不依赖抽卡。

| 行动 | 消耗 | 效果 |
| --- | ---: | --- |
| 普通攻击 | 1 行动力 | 使用当前武器造成基础伤害，并触发命中、职业资源和装备效果。 |
| 防御 | 1 行动力 | 获得护甲，并在本回合获得少量减伤。 |
| 躲避 | 1 行动力 | 获得 1 层躲避，抵消下一次直接攻击的 1 段伤害。 |
| 职业技能 | 0-3 行动力 | 使用当前 4 个技能槽中的技能，受冷却、资源和融合影响。 |
| 装备主动 | 0-2 行动力 | 触发装备主动效果，通常有每战限制或冷却。 |

## 普通攻击

普通攻击是稳定输出来源。

基础规则：

- 伤害由武器基础伤害、角色力量、装备词条和状态卡修饰共同决定。
- 普通攻击不会自然暴击；只有使用暴击状态卡、技能或套装提供的确定暴击效果后，下一次普通攻击或攻击技能才会暴击。
- 普通攻击可以触发战士怒气、弓箭手专注和装备命中特效。
- 普通攻击不能被状态卡替代；状态卡只能提高下一次或本回合攻击表现。

示例状态卡：

- 强击：下一次普通攻击伤害 +6。
- 穿刺角度：下一次普通攻击无视 20% 护甲。
- 致命瞄准：下一次普通攻击或攻击技能必定暴击。

## 防御

防御是稳定承伤手段。

基础规则：

- 防御行动获得护甲。
- 防御行动额外提供本回合 10% 减伤。
- 护甲优先抵消伤害。
- 回合结束时护甲默认清空，除非装备、技能或融合使其保留。

示例状态卡：

- 加固：下一次防御获得的护甲 +5。
- 卸力：本回合受到伤害降低 15%。
- 稳固站姿：若本回合使用过防御，下一次被攻击时额外获得 4 点护甲。

## 躲避

躲避是确定型规避手段，适合弓箭手和轻甲构筑。

基础规则：

- 躲避行动获得 1 层躲避。
- 每 1 层躲避抵消下一次直接攻击的 1 段伤害。
- 多段攻击会逐段消耗躲避层数。
- 躲避不能避免中毒、燃烧等持续伤害。
- 回合结束时未使用的躲避层默认清空，除非有装备、技能或套装使其保留。

示例状态卡：

- 侧步：获得 1 层躲避。
- 翻身卸力：本回合下一次直接伤害降低 20%。
- 反身射击：本回合消耗躲避层后，下一次普通攻击伤害 +8。

## 职业技能

每名角色同时装备 4 个技能。技能是职业战斗身份的核心，不进入状态卡抽取配置。

技能规则：

- 技能消耗行动力、职业资源或冷却。
- 技能可以被装备和融合增强；普通攻击、防御、躲避也可以承载本局内融合附着。
- 技能不会因为抽卡不顺而不可用，但可能受冷却或资源限制。
- 技能与普通攻击、防御、躲避共同构成完整战斗手段。

## 状态卡

状态卡是战斗中的临时修饰能力，不是玩家局外构筑的卡组。

状态卡规则：

- 每回合开始抽取 2 张。
- 每回合最多使用 2 张。
- 当前最多持有 5 张状态卡，超过上限时需要丢弃到已使用记录。
- 状态卡通常不直接造成大量伤害，不直接提供大量护甲。
- 状态卡主要修饰“下一次行动”或“本回合行动”。
- 强力状态卡可以有消耗、冷却或本场战斗限用。
- 状态卡只在当前战斗内存在；战斗结束后，当前持有、已使用和消耗记录全部清空。

状态卡抽取概率示例：

| 状态卡类型 | 基础抽取概率 |
| --- | ---: |
| 伤害修饰卡 | 45% |
| 防御/减伤卡 | 25% |
| 躲避卡 | 15% |
| 资源卡 | 10% |
| 暴击卡 | 5% |

抽取概率修正规则：

- 基础抽取概率是固定默认值。
- 装备、套装、技能、融合可以修改某一类状态卡的抽取概率。
- 暴击相关概率修正必须写成“暴击状态卡抽取概率 +X”，不代表攻击会自然暴击。
- 概率修正只影响状态卡抽取，不改变普通攻击、防御、躲避和技能的确定性结算。

状态卡战斗记录：

- 状态卡抽取配置：本场战斗可抽取的状态卡类型和概率。
- 当前持有：当前可使用的状态卡。
- 已使用记录：已使用或回合结束丢弃的状态卡。
- 消耗记录：本场战斗不会再次出现的状态卡。

## 回合流程

玩家回合开始：

1. 触发回合开始效果。
2. 获得 3 点行动力。
3. 抽 2 张状态卡。
4. 显示怪物意图。

玩家行动：

1. 使用状态卡修饰本回合行动。
2. 执行普通攻击、防御、躲避。
3. 使用 4 个技能槽中的职业技能。
4. 使用装备主动效果。
5. 结算融合附着触发。
6. 结束回合。

怪物回合：

1. 怪物按意图行动。
2. 结算躲避、护甲、减伤和反击。
3. 结算中毒、燃烧、流血等持续状态。
4. 清除临时护甲和本回合状态卡修饰。
5. 进入下一玩家回合。

## 怪物意图

怪物行动前必须向玩家展示意图。常见意图：

- 攻击：显示预计伤害和攻击次数。
- 防御：获得护甲或减伤。
- 强化：提高自身属性。
- 弱化：给玩家施加负面状态。
- 召唤：生成小怪。
- 特殊：触发怪物机制。

## 常用状态

正面状态：

| 状态 | 效果 |
| --- | --- |
| 力量 | 普通攻击和攻击技能伤害提高。 |
| 坚韧 | 防御获得的护甲提高。 |
| 专注 | 弓箭手职业资源。 |
| 怒气 | 战士职业资源。 |
| 减伤 | 受到伤害按固定百分比降低，不涉及随机判定。 |
| 躲避层 | 每层抵消下一次直接攻击的 1 段伤害。 |

负面状态：

| 状态 | 效果 |
| --- | --- |
| 破甲 | 受到攻击伤害提高。 |
| 标记 | 受到弓箭手相关伤害提高。 |
| 虚弱 | 造成伤害降低。 |
| 易伤 | 受到伤害提高。 |
| 中毒 | 回合开始受到持续伤害。 |

## 胜负条件

胜利：

- 当前战斗中所有敌人生命降至 0。
- 进入战斗胜利奖励界面。

失败：

- 玩家生命降至 0。
- 当前爬塔局结束。
- 清空塔内增强卡、Buff 卡和融合附着。
