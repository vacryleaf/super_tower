# 07 战斗系统

## 战斗类型

首版采用回合制卡牌战斗。玩家在自己的回合中抽牌、消耗能量打牌、使用技能和装备效果。玩家结束回合后，怪物按已显示意图行动。

## 基础参数

| 参数 | 默认值 |
| --- | ---: |
| 每回合抽牌 | 5 张 |
| 基础能量 | 3 点 |
| 起始手牌上限 | 10 张 |
| 每场战斗初始护甲 | 0 |
| 抽牌堆耗尽 | 洗入弃牌堆后继续抽牌 |

## 回合流程

玩家回合开始：

1. 触发回合开始效果。
2. 获得基础能量。
3. 抽 5 张牌。
4. 显示怪物意图。

玩家行动：

1. 打出手牌。
2. 使用装备主动效果。
3. 使用已装备技能。
4. 查看融合附着触发。
5. 结束回合。

怪物回合：

1. 怪物按意图行动。
2. 结算中毒、燃烧、流血等持续状态。
3. 清除临时护甲，除非有保留护甲效果。
4. 进入下一玩家回合。

## 卡牌区域

- 抽牌堆：未抽取卡牌。
- 手牌：当前可使用卡牌。
- 弃牌堆：本回合使用或回合结束丢弃的卡牌。
- 消耗区：本场战斗不会再次出现的卡牌。

## 怪物意图

怪物行动前必须向玩家展示意图。常见意图：

- 攻击：显示预计伤害。
- 防御：获得护甲或减伤。
- 强化：提高自身属性。
- 弱化：给玩家施加负面状态。
- 召唤：生成小怪。
- 特殊：触发怪物机制。

## 常用状态

正面状态：

| 状态 | 效果 |
| --- | --- |
| 力量 | 攻击伤害提高。 |
| 敏捷 | 护甲获得量提高。 |
| 专注 | 弓箭手职业资源。 |
| 怒气 | 战士职业资源。 |
| 保留 | 指定手牌回合结束不弃置。 |

负面状态：

| 状态 | 效果 |
| --- | --- |
| 破甲 | 受到攻击伤害提高。 |
| 标记 | 受到弓箭手相关伤害提高。 |
| 虚弱 | 造成伤害降低。 |
| 易伤 | 受到伤害提高。 |
| 中毒 | 回合开始受到持续伤害。 |

## 胜负条件

胜利：

- 当前战斗中所有敌人生命降至 0。
- 进入战斗胜利奖励界面。

失败：

- 玩家生命降至 0。
- 当前爬塔局结束。
- 清空塔内增强、融合和临时 Buff。
