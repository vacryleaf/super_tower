# 05 高塔与奖励

## 高塔结构

高塔层数无限递增。每层固定 10 场战斗，击败第 10 只怪物后进入下一层。塔层不提供固定终点，核心目标是刷新最高层数和构筑强度上限。

第 1 层是新手引导层，奖励固定，不进入正式奖励池。第 2 层开始使用普通怪 3 选 1、精英怪 4 选 1、Boss 三分支奖励结构。

## 新手引导层奖励

新手引导层固定解锁 9 件基础装备和第 1 个技能。项链和戒指不在新手引导层解锁。

| 战斗序号 | 类型 | 固定解锁 |
| --- | --- | --- |
| 1 | 普通怪 | 头部装备。 |
| 2 | 普通怪 | 上身装备。 |
| 3 | 普通怪 | 腰部装备。 |
| 4 | 强化怪 | 下身装备。 |
| 5 | 普通怪 | 手部装备。 |
| 6 | 普通怪 | 护腿装备。 |
| 7 | 普通怪 | 脚部装备。 |
| 8 | 精英怪 | 武器装备。 |
| 9 | 普通怪 | 副手装备。 |
| 10 | 新手 Boss | 第 1 个职业技能。 |

## 怪物成长

建议基础公式：

- 怪物生命 = 基础生命 * (1 + 当前层数 * 0.12) * 战斗序号强度系数。
- 怪物伤害 = 基础伤害 * (1 + 当前层数 * 0.08) * 战斗序号强度系数。
- 怪物机制复杂度每 5 层提升一次。

深层调节：

- 第 1-5 层：帮助玩家理解基础系统。
- 第 6-15 层：要求玩家形成明确构筑。
- 第 16-30 层：强化怪和精英怪开始针对单一策略。
- 第 31 层以后：数值与机制并行上升，非装备卡进入高稀有度循环，Boss 永久装备优先补全接近下一档能力的套装。

## 正式胜利奖励

第 2 层开始，普通怪和强化怪胜利后展示 3 个非装备卡奖励选项，玩家选择 1 个：

- 技能卡
- 增强卡
- Buff 卡
- 状态卡概率强化
- 金币
- 治疗

精英怪胜利后展示 4 个非装备卡奖励选项，玩家选择 1 个。Boss 胜利后提供 3 类奖励分支：

| 分支 | 选项数量 | 规则 |
| --- | ---: | --- |
| 非装备卡奖励 | 5 选 1 | 使用普通怪的非装备卡奖励池，并提高高稀有度候选优先级。 |
| 永久装备 | 1 件 | 解锁 1 件永久装备，加入角色永久装备库。 |
| 技能奖励 | 1 个 | 优先从未拥有永久技能中生成；永久技能全解锁后固定改为塔内技能。 |

技能奖励去重规则：

- 永久技能不会重复获取。
- 塔内技能不会与本局已装备技能重复。
- 塔内技能不会与本局候选技能列表中的同名技能重复。
- 若无可用塔内技能，奖励生成器改为提供高品质增强卡。

奖励选项生成时应考虑：

- 当前职业。
- 当前已装备技能标签。
- 当前层数。
- 上一场战斗类型。

装备卡获取限制：

- 普通怪、强化怪和精英怪不生成装备卡。
- Boss 的 5 选 1非装备卡奖励不生成装备卡。
- 装备卡只在 Boss 永久装备分支中出现。
- 第 1 层新手引导的装备是固定系统解锁，不进入正式装备卡奖励池。

## 奖励生成

卡牌稀有度权重用于技能卡、增强卡、Buff 卡、状态卡概率强化和融合词条，不用于装备强弱。正式装备卡只在 Boss 永久装备分支中生成。

Boss 永久装备候选规则：

| 候选情况 | 处理规则 |
| --- | --- |
| 项链或戒指 | 只生成套装装备。 |
| 已有套装接近下一档能力 | 优先提供缺失部件。 |
| 非项链、非戒指栏位 | 可以生成普通装备或新的套装起点。 |

当玩家已经拥有某套装的 1/2/3/4/5 件时，Boss 永久装备分支优先提供能触发下一档套装能力的缺失部件。

非装备卡基础稀有度权重：

普通怪基础权重：

| 稀有度 | 权重 |
| --- | ---: |
| 普通 | 65 |
| 优秀 | 25 |
| 稀有 | 9 |

| --- | ---: |
| 普通 | 0 |
| 优秀 | 50 |
| 稀有 | 35 |

层末首领基础权重：

| 稀有度 | 权重 |

| 普通 | 0 |

每 10 层提高稀有、史诗和传说权重，降低普通权重。

Boss 的 5 选 1非装备卡奖励使用层末首领稀有度权重。精英怪的 4 选 1非装备卡奖励使用精英怪稀有度权重。

## 特殊节点

首版可以先不做复杂路线选择，但建议预留节点类型：

- 商店：用金币购买技能、增强卡、Buff 卡、状态卡概率强化和融合材料；不出售装备卡。
- 铁匠：强化装备卡。
- 祭坛：以生命或已有卡牌换取高品质奖励。
- 休整：治疗或移除负面状态。
