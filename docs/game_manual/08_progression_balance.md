# 08 成长与数值平衡

## 成长边界

永久成长应保持克制，局内成长应足够强。核心体验来自单局构筑，而不是永久数值碾压。

永久内容：

- 职业解锁。
- 基础卡牌解锁。
- 基础装备库扩展。
- Boss 奖励解锁的永久装备。
- Boss 奖励解锁的职业永久技能。
- 图鉴、成就、最高层数。
- 少量开局选项。

局内内容：

- 战斗胜利奖励。
- 装备替换。
- 技能替换。
- 塔内附着卡形成的融合附着。
- 装备、技能和基础行动上的局内附着。

## 角色基础成长

首版建议不做等级制永久加点。角色可保留历史记录，但每局开局属性主要由职业和起始配置决定。

若需要轻量永久成长，建议只开放横向内容：

- 解锁新技能池。
- 解锁新装备掉落。
- 解锁新开局遗物。
- 解锁新怪物图鉴说明。

职业永久技能获取必须去重。Boss 技能奖励优先从未拥有的职业永久技能池中抽取；当该职业永久技能池全部解锁后，该奖励位固定转为塔内技能，继续服务当前局构筑。首版塔内技能池优先使用通用塔内技能，通用塔内技能不进入永久成长。

## 怪物数值公式

普通怪：

```text
生命 = 基础生命 * (1 + 层数 * 0.12) * 战斗序号系数
伤害 = 基础伤害 * (1 + 层数 * 0.08) * 战斗序号系数
```

精英怪：

```text
生命 = 普通怪生命 * 1.45
伤害 = 普通怪伤害 * 1.25
机制强度 = 普通怪机制 + 1 个构筑针对机制
```

首领：

```text
生命 = 普通怪生命 * 2.2
伤害 = 普通怪伤害 * 1.35
机制强度 = 当前层主题机制 + 1 个阶段转换机制
```

## 奖励权重调节

奖励应服务当前构筑，而不是脱离玩家当前构筑。

建议奖励池由三部分组成：

| 来源 | 权重 | 说明 |
| --- | ---: | --- |
| 职业相关 | 40% | 与战士怒气、弓箭手专注等职业机制相关。 |
| 当前构筑相关 | 35% | 根据已装备技能和装备标签生成。 |
| 探索转型 | 25% | 提供转型和惊喜空间。 |

## 融合强度预算

融合目标不设置附着数量上限。装备、已装备技能、普通攻击、防御和躲避都按独立目标计算，因此平衡不能依赖槽位限制，必须依赖单条附着强度、触发条件、稀有度和奖励出现节奏。

塔内附着卡的固定数值按照获得塔层提高，避免高层奖励仍停留在低层数值。数值在获得时锁定，不随玩家继续爬塔自动成长。首版使用以下等级：

```text
S = max(0, floor((获得塔层 - 1) / 10))
```

| 字段 | 公式 | 说明 |
| --- | --- | --- |
| A | 3 + S | 攻击强化。 |
| D | 3 + S | 护甲强化。 |
| H | 3 + S | 生命强化。 |
| G | 1 + floor(S / 3) | 越战越强每次命中获得的临时攻击。 |
| B | 6 + 2S | 护体启动。 |
| R | 5 + S | 再生。 |

百分比、倍率、触发次数、移除层数、破甲层数和保命类规则默认不随塔层成长，避免高层机制效果失控。

建议规则：

- 普通附着只加基础数值或简单触发。
- 优秀附着增加条件收益。
- 稀有附着改变触发频率。
- 史诗附着改变打法。
- 传说附着允许构筑核心效果，但应带条件或代价。

## 难度曲线目标

| 层数 | 目标体验 |
| --- | --- |
| 1-3 | 学会基础战斗和奖励选择。 |
| 4-10 | 形成第一个明确流派。 |
| 11-20 | 要求装备、技能、融合三者联动。 |
| 21-40 | 需要处理反制机制和资源压力。 |
| 41+ | 无限挑战，依靠构筑极限和玩家决策。 |
