# 08 成长与数值平衡

## 成长边界

永久成长应保持克制，局内成长应足够强。核心体验来自单局构筑，而不是永久数值碾压。

永久内容：

- 职业解锁。
- 基础卡牌解锁。
- 基础装备库扩展。
- Boss 奖励解锁的永久装备。
- Boss 奖励解锁的永久技能。
- 图鉴、成就、最高层数。
- 少量开局选项。

局内内容：

- 战斗胜利奖励。
- 装备替换。
- 技能替换。
- 增强卡。
- Buff 卡。
- 融合附着。

## 角色基础成长

首版建议不做等级制永久加点。角色可保留历史记录，但每局开局属性主要由职业和起始配置决定。

若需要轻量永久成长，建议只开放横向内容：

- 解锁新技能池。
- 解锁新装备掉落。
- 解锁新开局遗物。
- 解锁新怪物图鉴说明。

永久技能获取必须去重。Boss 技能奖励优先从未拥有的永久技能池中抽取；当永久技能池全部解锁后，该奖励位固定转为塔内技能，继续服务当前局构筑。

## 怪物数值公式

普通怪：

```text
生命 = 基础生命 * (1 + 层数 * 0.12) * 战斗序号系数
伤害 = 基础伤害 * (1 + 层数 * 0.08) * 战斗序号系数
```

精英怪：

```text
生命 = 普通怪生命 * 1.45
伤害 = 普通怪伤害 * 1.25
机制强度 = 普通怪机制 + 1 个构筑针对机制
```

首领：

```text
生命 = 普通怪生命 * 2.2
伤害 = 普通怪伤害 * 1.35
机制强度 = 当前层主题机制 + 1 个阶段转换机制
```

## 奖励权重调节

奖励应服务当前构筑，而不是脱离玩家当前构筑。

建议奖励池由三部分组成：

| 来源 | 权重 | 说明 |
| --- | ---: | --- |
| 职业相关 | 40% | 与战士怒气、弓箭手专注等职业机制相关。 |
| 当前构筑相关 | 35% | 根据已装备技能和装备标签生成。 |
| 探索转型 | 25% | 提供转型和惊喜空间。 |

## 融合强度预算

每个目标最多 3 个融合附着，因此单条融合可以强，但必须避免无条件指数叠加。

建议规则：

- 普通增强只加基础数值。
- 优秀增强增加条件收益。
- 稀有增强改变触发频率。
- 史诗增强改变打法。
- 传说增强允许构筑核心效果，但应带条件或代价。

## 难度曲线目标

| 层数 | 目标体验 |
| --- | --- |
| 1-3 | 学会基础战斗和奖励选择。 |
| 4-10 | 形成第一个明确流派。 |
| 11-20 | 要求装备、技能、融合三者联动。 |
| 21-40 | 需要处理反制机制和资源压力。 |
| 41+ | 无限挑战，依靠构筑极限和玩家决策。 |
