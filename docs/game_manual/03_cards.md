# 03 卡牌系统

## 卡牌类型

首版包含 4 类核心卡牌：

| 类型 | 定义 | 主要用途 |
| --- | --- | --- |
| 装备卡 | 可装入装备栏的卡牌 | 提供属性、被动、主动或触发效果。 |
| 技能卡 | 职业技能或通用技能 | 装入 4 个技能槽，或作为融合材料。 |
| 增强卡 | 局内强化材料 | 强化装备、技能或战斗卡牌的数值和机制。 |
| Buff 卡 | 临时增益或规则改变 | 可直接生效，也可融合附着到装备或技能。 |

## 稀有度

| 稀有度 | 颜色建议 | 作用 |
| --- | --- | --- |
| 普通 | 白色 | 基础数值与简单效果。 |
| 优秀 | 绿色 | 带有 1 个明确构筑方向。 |
| 稀有 | 蓝色 | 提供强力组合效果或条件收益。 |
| 史诗 | 紫色 | 改变玩法节奏，适合作为构筑核心。 |
| 传说 | 金色 | 高风险高收益，通常来自精英、首领或深层奖励。 |

## 标签

标签用于奖励筛选、融合限制和效果联动。

通用标签：

- 攻击
- 防御
- 恢复
- 抽牌
- 能量
- 暴击
- 破甲
- 标记
- 怒气
- 专注
- 召回
- 连击

装备标签：

- 重甲
- 轻甲
- 近战
- 远程
- 盾牌
- 双持
- 饰品
- 触发

## 装备卡

装备卡必须绑定一个装备栏位。玩家获得装备卡后可以：

- 立即装备到对应栏位。
- 保留在本局背包。
- 分解为金币或融合碎片。
- 作为融合目标承载增强。

装备卡可提供：

- 固定属性：生命、护甲、攻击、暴击率。
- 被动效果：每回合开始获得护甲。
- 主动效果：消耗能量触发。
- 条件触发：暴击后抽牌、受到伤害后反击。

## 技能卡

技能卡分为职业专属和通用技能。玩家在塔内获得技能卡后可以：

- 装入空技能槽。
- 替换已装备技能。
- 保留为本局候选技能。
- 作为融合材料附着到装备或其他技能。

同一角色最多同时装备 4 个技能。技能槽是构筑上限，不随层数自然增加。

## 增强卡

增强卡用于提高装备或技能的强度。增强卡本身不进入战斗抽牌堆，必须选择目标后才能生效。

增强示例：

- 伤害 +4
- 护甲 +5
- 消耗 -1，最低为 0
- 每场战斗首次触发时额外抽 1 张牌
- 触发条件从暴击改为命中

增强卡只在当前爬塔局内有效。

## Buff 卡

Buff 卡可以直接使用，也可以作为融合材料。

直接使用时：

- 立即获得整场战斗、若干场战斗或整局有效的临时 Buff。
- 不占用装备栏和技能槽。

融合使用时：

- 需要选择装备或已装备技能作为目标。
- 转化为目标上的 1 个融合附着。
- 占用目标的融合附着上限。
