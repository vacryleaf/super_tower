# 06 融合系统

## 设计目标

融合系统让玩家把战斗奖励转化为当前构筑的定向强化。它是局内成长的核心，但不会跨局继承。

## 可融合对象

可作为融合目标：

- 已装备的装备卡。
- 当前装备在 4 个技能槽中的技能卡。
- 基础行动：普通攻击、防御、躲避。

不可作为融合目标：

- 未装备技能。
- 未装备装备。
- 纯金币奖励。
- 一次性治疗。

## 可作为融合材料的卡牌

- 塔内附着卡。
- 技能卡。

塔内附着卡不能直接使用，必须在获得时立即附着到装备、已装备技能或基础行动上。附着后本局永久生效，跨局不继承。

装备卡默认不能作为融合材料。若后续增加装备吞噬系统，应作为独立功能，不混入首版融合规则。

## 附着数量

融合目标不设置附着数量上限。已装备的装备、已装备技能、普通攻击、防御和躲避都可以承载任意数量的融合附着。

融合附着包含：

- 来源卡牌名称。
- 来源卡牌类型。
- 效果描述。
- 稀有度。
- 标签。
- 当前局 ID。

## 融合流程

1. 玩家在奖励界面选择塔内附着卡或技能卡。
2. 系统进入融合目标选择状态。
3. 高亮可融合的装备、已装备技能和基础行动。
4. 玩家选择目标。
5. 融合立即生效，并保存到当前局内状态。
6. 塔内附着卡若未完成附着，则不能领取该奖励，也不能进入下一场战斗。

## 融合效果类型

首版塔内附着卡只使用以下 20 种效果，并统一按同一套附着规则结算。

| 编号 | 卡牌 | 效果 |
| ---: | --- | --- |
| 1 | 攻击强化 | 附着目标攻击 +3。 |
| 2 | 攻击增幅 | 附着目标攻击 * 1.1。 |
| 3 | 护甲强化 | 附着目标护甲 +3。 |
| 4 | 护甲增幅 | 附着目标护甲 * 1.1。 |
| 5 | 生命强化 | 生命上限 +3。 |
| 6 | 生命增幅 | 生命上限 * 1.1。 |
| 7 | 裂甲追击 | 附着目标命中有破甲的敌人时，追加 1 次低效攻击，每次行动最多 1 次。 |
| 8 | 处决追击 | 附着目标击败敌人后，追加 1 次低效攻击，可重新选择目标。 |
| 9 | 吸血 | 附着目标造成攻击伤害后，按实际伤害的 10% 恢复生命。 |
| 10 | 碎甲 | 附着目标每次攻击命中时，施加 1 层破甲。 |
| 11 | 越战越强 | 附着目标每次攻击命中后，本场战斗内攻击 +1，可叠加，战斗结束清空本条产生的临时层数。 |
| 12 | 共鸣锁定 | 目标上其他附着的数值效果提高 25%；该目标之后不能替换、移除或重铸已有附着，仍可继续新增附着。 |
| 13 | 净化攻击 | 附着目标攻击后，移除自身 1 层负面状态。 |
| 14 | 护体启动 | 附着目标每场战斗首次触发或被使用时，获得 6 点护甲。 |
| 15 | 余势 | 附着目标击败敌人后，下一次基础行动效果 +30%。 |
| 16 | 急迫 | 生命低于 40% 时，附着目标每场战斗首次触发不消耗行动力。 |
| 17 | 破势 | 附着目标命中特殊意图敌人时，该意图效果降低 30%。 |
| 18 | 再生 | 只要附着目标有效，每个玩家回合开始恢复 5 点生命。 |
| 19 | 致命感知 | 暴击状态卡抽取概率 +5%；不提供角色天然暴击率。 |
| 20 | 命悬一线 | 每场战斗首次受到致死伤害时，生命保留为 1。 |

数值结算顺序：

1. 先结算固定值，例如攻击 +3、护甲 +3、生命上限 +3。
2. 再结算倍率，例如攻击 * 1.1、护甲 * 1.1、生命上限 * 1.1。
3. 最后结算共鸣锁定的其他附着数值效果 +25%。
4. 同类倍率多次出现时连乘，最终数值按战斗系统统一取整。

技能转化：

- 技能卡可将自身标签或部分效果转化为附着。
- 示例：将「猎人标记」融合到弓上，使该弓每场战斗首次命中时施加 1 层标记。
- 示例：将「盾墙」融合到胸甲上，使每场战斗开始时获得 6 点护甲。

## 跨局规则

融合只属于当前爬塔局：

- 开启新游戏时清空所有融合附着。
- 死亡后清空所有融合附着。
- 主动重开时清空所有融合附着。
- 永久装备库不记录融合附着。
- 基础行动上的融合附着也只记录在当前局内状态中，不写入角色永久数据。

这条规则是核心边界，任何存档、UI 和数据结构都必须明确区分永久装备与局内装备实例。
