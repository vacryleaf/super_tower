# 14 敌人设计

## 设计目标

首版敌人单位池按普通、精英、Boss 为 7:2:1 设计，共 50 种敌人单位。实际战斗从“战斗编队”中生成，部分普通、精英和 Boss 战斗会包含多个敌人。

| 类型 | 数量 | 单体规则 |
| --- | ---: | --- |
| 普通敌人 | 35 | 拥有 1 种特性；部分普通敌人拥有 1 个技能。 |
| 精英敌人 | 10 | 拥有 2 种特性；拥有 1-2 个技能。 |
| Boss | 5 | 拥有 3 种特性；拥有 3 个技能。 |

每层塔固定 10 场战斗，首版默认节奏为 7 场普通、2 场精英、1 场 Boss。每场战斗可以包含 1 个或多个敌人；多敌人战斗中的单个敌人比同级单体战更弱，但整场战斗总强度略高。玩家理论上先手；拥有“先手”特性的敌人会在第 1 回合玩家行动前先结算一次敌方行动。

初次开发不依赖美术资源。敌人以卡牌、名称、文本描述、意图图标和数值条呈现。战斗视角采用第一人称打牌视角：敌人卡牌排列在屏幕上方或中上部，玩家行动、状态卡、技能和装备按钮在下方。

## 楼层数值公式

敌人生命、攻击、防御都随层数成长。设：

```text
F = 当前塔层，最低为 1
post_gate = max(0, F - 5)
growth = 1 + 0.10 * (F - 1) + 0.52 * post_gate + 0.18 * post_gate² + 0.25 * floor((F - 1) / 10)
```

普通敌人基础值：

```text
普通生命 = round((22 + 4.6F) * growth * 生命系数)
普通攻击 = round((4.2 + 1.0F) * growth * 攻击系数)
普通防御 = round((1.6 + 0.5F) * growth * 防御系数)
```

精英敌人基础值：

```text
精英生命 = 普通生命 * 2.0
精英攻击 = 普通攻击 * 1.2
精英防御 = 普通防御 * 1.3
```

Boss 基础值：

```text
Boss 生命 = 普通生命 * 3.4
Boss 攻击 = 普通攻击 * 1.4
Boss 防御 = 普通防御 * 1.8
```

这些公式会让高层敌人数值持续抬升，并每 10 层出现一次明显台阶。只有构筑成型、技能与装备联动稳定、塔内附着选择合理的角色，才能继续推进到更高层。

## 多敌人编队规则

敌人单位使用上方公式先计算单体标准值，再根据编队人数和单位定位应用编队系数。编队系数只影响该场战斗，不改变图鉴中的敌人单位基础系数。

| 编队人数 | 单个普通单位系数 | 单个精英单位系数 | Boss 本体系数 | 整场目标强度 |
| ---: | ---: | ---: | ---: | --- |
| 1 | 1.00 | 1.00 | 1.00 | 标准单体战。 |
| 2 | 0.62 | 0.72 | 0.90 | 总生命/总伤害约为单体战 120%-140%。 |
| 3 | 0.48 | 0.58 | 0.82 | 总生命/总伤害约为单体战 135%-160%。 |
| 4 | 0.38 | 0.48 | 0.76 | 总生命/总伤害约为单体战 145%-170%。 |

多敌人战斗结算规则：

- 击败该场全部敌人才算完成 1 场战斗。
- 多敌人战斗仍只发放 1 次胜利奖励。
- 同一场战斗最多 1 个敌人拥有“先手”；若编队中多个敌人有先手，只由最靠前的敌人执行先手。
- 多敌人战斗中，每回合最多 2 个敌人执行攻击意图，其余敌人优先执行防御、蓄力、标记、召唤或辅助意图。
- 召唤类技能产生的临时单位不计入本层 10 场战斗进度，也不额外发放奖励。
- 编队里如果出现“哥布林小队”这类低级单位，单个哥布林必须明显弱于普通单体敌人，但三人合计的压力略高于普通单体战。

## 敌人特性

| 特性 | 效果 |
| --- | --- |
| 先手 | 第 1 回合玩家行动前，敌人先执行 1 次基础意图。 |
| 厚皮 | 战斗开始获得防御值 2 倍的护甲。 |
| 利爪 | 攻击意图造成的伤害提高 15%。 |
| 群袭 | 攻击意图拆分为 2 段伤害。 |
| 腐蚀 | 攻击命中后施加 1 层腐蚀，使目标下次获得护甲降低 20%。 |
| 破甲 | 攻击命中后施加 1 层破甲。 |
| 警觉 | 玩家每回合使用第 2 张状态卡后，敌人获得防御值的护甲。 |
| 复苏 | 每 3 回合恢复 5% 最大生命。 |
| 狂暴 | 生命低于 40% 后，攻击意图伤害提高 30%。 |
| 护卫 | 同场其他敌人受到攻击时，本敌人获得防御值的护甲。 |
| 闪身 | 每 2 回合获得 1 层躲避。 |
| 标记 | 攻击命中后施加 1 层标记，后续攻击标记目标伤害提高。 |
| 汲血 | 攻击造成伤害后，按实际伤害 20% 恢复生命。 |
| 反制 | 玩家连续两次使用攻击行动后，敌人下一次攻击附带破甲。 |
| 召唤 | 每 4 回合召唤一个同族低级单位；首版可用文本单位占位。 |
| 法盾 | 每场战斗首次受到技能伤害时，减少 50% 该次伤害。 |
| 诅咒 | 每 3 回合将 1 张负面状态加入玩家当前持有状态卡。 |
| 充能 | 每次防御后获得 1 层充能；3 层后下一次攻击提高。 |
| 固守 | 若上回合没有受到生命伤害，本回合防御意图翻倍。 |
| 裂变 | 死亡时对玩家造成攻击值 50% 的伤害。 |

## 生成与难度规则

每层战斗槽：

| 战斗序号 | 敌人类型 |
| ---: | --- |
| 1 | 普通敌人 |
| 2 | 普通敌人 |
| 3 | 普通敌人 |
| 4 | 精英敌人 |
| 5 | 普通敌人 |
| 6 | 普通敌人 |
| 7 | 普通敌人 |
| 8 | 精英敌人 |
| 9 | 普通敌人 |
| 10 | Boss |

普通敌人生成：

1. 选择当前层数允许出现的族群。
2. 在普通编队池中抽取 1 个编队。
3. 第 1-5 层避免连续出现召唤、裂变和 3 人以上编队。

精英战斗生成：

1. 每层第 4 场和第 8 场优先生成精英战斗。
2. 在精英编队池中抽取 1 个编队。
3. 第 10 层以前精英编队优先单体或 2 人组；第 10 层以后允许 3 人精英编队。

Boss 战生成：

1. 每层第 10 场为 Boss。
2. 在 Boss 编队池中抽取或按族群轮换 1 个编队。
3. Boss 的 3 个技能按意图轮转，但生命阈值技能优先于普通轮转。

难度递进：

- 第 1-5 层：只出现低复杂度普通敌人和弱化 Boss，帮助玩家理解基础行动。
- 第 6-15 层：开始出现先手、标记、破甲、护卫等构筑压力。
- 第 16-30 层：精英技能和 Boss 机制开始要求玩家处理召唤、法盾和多段伤害。
- 第 31 层以后：敌人数值和机制同时提高，只有成功构筑才能稳定推进。
