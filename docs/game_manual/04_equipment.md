# 04 装备系统

## 装备栏位

角色拥有 11 个装备栏：

| 栏位 | 说明 | 常见属性 |
| --- | --- | --- |
| 头部 | 头盔、兜帽、面具 | 生命、抗性、状态卡抽取。 |
| 上身 | 胸甲、皮甲、法袍 | 生命、护甲、减伤。 |
| 腰部 | 腰带、束带 | 药剂、金币、生命。 |
| 下身 | 裤甲、护裙 | 护甲、闪避、移动类触发。 |
| 手部 | 手套、护腕 | 攻击、暴击状态卡、技能强化。 |
| 护腿 | 腿甲、绑腿 | 护甲、反击、行动收益。 |
| 脚部 | 靴子、战靴 | 躲避、先手、状态卡抽取。 |
| 武器 | 剑、斧、弓、弩 | 伤害、职业联动。 |
| 副手 | 盾、箭袋、副武器 | 防御、弹药、触发效果。 |
| 项链 | 项链、护符 | 资源、特殊效果。 |
| 戒指 | 戒指 | 构筑核心、特殊触发。 |

首版戒指栏按 1 个戒指位设计。后续版本可扩展为 2 个戒指位，但首版 UI 和数值不依赖双戒指。

## 装备卡规则

- 每个装备栏最多装备 1 张装备卡。
- 装备卡必须与栏位匹配。
- 职业不强制限制装备，但不同职业对装备标签有收益倾向。
- 装备不分强弱稀有度，只分普通装备与套装。
- 普通装备提供更高基础数值，适合补强短板或不追求套装的构筑。
- 套装装备基础数值低于同栏位普通装备，但通过套装件数提供阶段性特殊能力。
- 项链和戒指没有基础普通装备，只能作为套装装备出现。
- 替换装备时，新装备继承自身基础属性，不继承旧装备上的融合。
- 被替换装备若仍在本局背包中，可保留其融合；若分解，则其融合一并消失。

## 普通装备与套装

普通装备：

- 不属于任何套装。
- 单件基础数值高于套装装备。
- 效果直接生效，不需要凑件。
- 适合前中期稳定过渡，或用于填补生命、防御、伤害、躲避等短板。
- 不包含项链和戒指栏位。

套装装备：

- 拥有套装 ID，例如 `warrior_iron_vanguard` 或 `archer_shadow_hunter`。
- 可分为职业套装和通用套装；通用套装不限职业。
- 单件基础数值低于普通装备。
- 装备同一套装达到指定件数时解锁套装能力。
- 首版支持 2/3/4/5/6 件套，其中 6 件套必须包含 2/4/6 三档能力。
- 项链和戒指只能通过套装装备进入游戏。

套装解锁规则：

| 套装规模 | 解锁效果 |
| ---: | --- |
| 2 件套 | 凑齐 2 件时解锁 1 档能力。 |
| 3 件套 | 凑齐 3 件时解锁 1 档能力。 |
| 4 件套 | 2 件时解锁第一档，4 件时保留 2 件效果并解锁第二档。 |
| 5 件套 | 2 件时解锁第一档，5 件时保留 2 件效果并解锁第二档。 |
| 6 件套 | 2/4/6 件分别解锁一档，同时拥有三档能力。 |

若玩家装备超过 6 件同套装装备，只按 6 件套结算，不额外叠加。不同套装可以同时生效，例如 2 件战士防御套 + 2 件战士怒气套，或 2 件弓箭手标记套 + 2 件弓箭手躲避套。

首版完整基础装备、套装清单与生成规则见 [12_equipment_generation.md](12_equipment_generation.md)。
套装装备的逐件装备名、栏位和基础属性见 [13_set_equipment_items.md](13_set_equipment_items.md)。

## 装备效果类型

固定属性：

- 最大生命 +10
- 每回合初始护甲 +4
- 暴击率 +5%，等同于暴击状态卡抽取概率 +5%
- 普通攻击和攻击技能伤害 +2

被动效果：

- 每场战斗开始时获得 1 点怒气。
- 每回合第一次执行防御行动时抽 1 张状态卡。
- 若生命低于 30%，获得额外护甲。

主动效果：

- 消耗 1 行动力，获得 12 点护甲。
- 消耗 2 行动力，对目标造成武器伤害。
- 每场战斗限用 1 次，恢复 15 点生命。

触发效果：

- 使用暴击状态卡并命中时获得 1 点专注。
- 受到未格挡伤害后，对敌人施加 1 层破甲。
- 本回合第 3 次攻击命中时追加一次小额伤害。

## 职业装备倾向

战士倾向：

- 武器：剑、斧、锤。
- 副手：盾牌。
- 防具：重甲。
- 关键词：护甲、怒气、反击、破甲、生命恢复。

弓箭手倾向：

- 武器：弓、弩。
- 副手：箭袋、短刃。
- 防具：轻甲。
- 关键词：标记、专注、暴击状态卡、连击、躲避、状态卡抽取。
