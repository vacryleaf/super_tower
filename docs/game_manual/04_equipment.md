# 04 装备系统

## 装备栏位

角色拥有 11 个装备栏：

| 栏位 | 说明 | 常见属性 |
| --- | --- | --- |
| 头部 | 头盔、兜帽、面具 | 生命、抗性、抽牌。 |
| 上身 | 胸甲、皮甲、法袍 | 生命、护甲、减伤。 |
| 腰部 | 腰带、束带 | 药剂、金币、生命。 |
| 下身 | 裤甲、护裙 | 护甲、闪避、移动类触发。 |
| 手部 | 手套、护腕 | 攻击、暴击、技能强化。 |
| 护腿 | 腿甲、绑腿 | 护甲、反击、行动收益。 |
| 脚部 | 靴子、战靴 | 闪避、先手、抽牌。 |
| 武器 | 剑、斧、弓、弩 | 伤害、职业联动。 |
| 副手 | 盾、箭袋、副武器 | 防御、弹药、触发效果。 |
| 项链 | 项链、护符 | 资源、稀有效果。 |
| 戒指 | 戒指 | 构筑核心、特殊触发。 |

首版戒指栏按 1 个戒指位设计。后续版本可扩展为 2 个戒指位，但首版 UI 和数值不依赖双戒指。

## 装备卡规则

- 每个装备栏最多装备 1 张装备卡。
- 装备卡必须与栏位匹配。
- 职业不强制限制装备，但不同职业对装备标签有收益倾向。
- 替换装备时，新装备继承自身基础属性，不继承旧装备上的融合。
- 被替换装备若仍在本局背包中，可保留其融合；若分解，则其融合一并消失。

## 装备效果类型

固定属性：

- 最大生命 +10
- 每回合初始护甲 +4
- 暴击率 +5%
- 攻击牌伤害 +2

被动效果：

- 每场战斗开始时获得 1 点怒气。
- 每回合第一次打出防御牌时抽 1 张牌。
- 若生命低于 30%，获得额外护甲。

主动效果：

- 消耗 1 能量，获得 12 点护甲。
- 消耗 2 能量，对目标造成武器伤害。
- 每场战斗限用 1 次，恢复 15 点生命。

触发效果：

- 暴击时获得 1 点专注。
- 受到未格挡伤害后，对敌人施加 1 层破甲。
- 打出第 3 张攻击牌时追加一次小额伤害。

## 职业装备倾向

战士倾向：

- 武器：剑、斧、锤。
- 副手：盾牌。
- 防具：重甲。
- 关键词：护甲、怒气、反击、破甲、生命恢复。

弓箭手倾向：

- 武器：弓、弩。
- 副手：箭袋、短刃。
- 防具：轻甲。
- 关键词：标记、专注、暴击、连击、抽牌。
